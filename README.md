# ReLF SEED-SD Reproduction

This project aims to reproduce and understand the ReLF model from the paper:

"Investigating the Effects of Sleep Conditions on Emotion Responses with EEG Signals and Eye Movements"

Main goals:
- Understand the SEED-SD dataset structure
- Build a memory-efficient data inspection pipeline
- Implement baseline models
- Re-implement ReLF step by step
- Document the full process for thesis and portfolio purposes

Dataset location:
- Local only: `D:/SEED-SD`
- Dataset files are not tracked by Git.
